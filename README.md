# Lab 2 Cloud Calculations

FTP-клиент для работы с облачными вычислениями и файловыми операциями.

## Описание

Приложение представляет собой графический FTP-клиент, разработанный на Java с использованием Swing. Позволяет подключаться к FTP-серверу, просматривать файлы, передавать их между локальным компьютером и сервером.

## Функциональность

- **Подключение к FTP-серверу** — ввод адреса, порта, логина и пароля
- **Просмотр файлов** — отображение списка файлов и папок локально и на сервере
- **Навигация по каталогам** — переход по папкам с отображением истории
- **Передача файлов** — загрузка и скачивание файлов между локальным ПК и сервером
- **Автоматическое создание структуры** — при первом подключении создаётся необходимая структура каталогов

## Технологии

- **Java 21**
- **Apache Commons Net 3.10.0** — для FTP-операций
- **Swing** — графический интерфейс

## Сборка и запуск

### Требования

- JDK 21 или выше
- Maven 3.x

### Сборка

```bash
mvn clean package
```

### Запуск

```bash
mvn exec:java -Dexec.mainClass="github.androkc.lab_2_cloud_calculations.Main"
```

Или через скомпилированный JAR:

```bash
java -jar target/lab_2_cloud_calculations-1.0-SNAPSHOT.jar
```

## Использование

1. Запустите приложение
2. Введите параметры подключения к FTP-серверу:
   - Адрес сервера
   - Порт (по умолчанию 21)
   - Имя пользователя
   - Пароль
3. Нажмите "Подключиться"
4. Используйте панели для навигации по файлам:
   - Левая панель — локальные файлы
   - Правая панель — файлы на сервере
5. Для передачи файлов используйте кнопки или двойной клик
